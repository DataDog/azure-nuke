# Azure AD User

## Details

- **Type:** `AzureADUser`
- **Scope:** tenant

## Properties

- **`Name`**: The DisplayName of the Entra ID User
- **`UPN`**: This is the user principal name of the Entra ID user, usually in the format of email
- **`ID`**: The ID of the Entra ID User
